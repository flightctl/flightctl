FlightCtl RPM Repository

This is the RPM repository for FlightCtl.

Usage

Add the repository:

sudo dnf config-manager --add-repo https://flightctl.github.io/flightctl/flightctl.repo

Install FlightCtl packages:

sudo dnf install flightctl

Repository Files

  • flightctl.repo - Repository configuration file
  • rpms/ - RPM packages directory
  • repodata/ - Repository metadata

Repository automatically updated from COPR builds.
